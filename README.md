**This repo has scripts used for Omni**.

For profile generation, run the script **pcfGen.py** in the omni-pages-generation folder.  
This script uses the template **template-pcfGen.txt** which is the updated working template.  

Similarly run **thesispcfGen** using **template-thesisGen.txt** file.

This will create pcfs in **projects/sesh folder**.  
Follow the documentation in the code for more information.
